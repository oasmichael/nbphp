nbphp 1.0
=====
nbphp.so-php日志扩展<br />
由强哥一手打造<br />

Linux 安装方式<br />
phpize<br />
./configure --with-php-config=xxxx(php-config的路径)<br />
make<br />
make install<br />

ps:mac系统需要加sudo你懂的<br />

功能介绍<br />
支持PHP各类型变量记录日志(字符串、数组、对象、资源等)<br />

支持函数列表<br />
nl(log,cat,logpath) 日志,分类,日志目录(后两个参数是可选参数)<br />
